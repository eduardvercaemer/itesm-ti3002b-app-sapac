# Developing

Install [bun](https://bun.sh/)

```shell
# install dependencies
bun install
# run dev server (open localhost:5173)
bun run dev
# format code
bun run format
```

# Modules

## Handkey Module

* `useEmployee`: Load employee data from ID.
* `useEmployeeList`: Get list of employee IDs.
